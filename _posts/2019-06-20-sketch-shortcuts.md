从 Photoshop 转为使用 Sketch 已经两个月了，体验非常优秀，Sketch 已经成为运营工作设计任务的主力工具。整理了一些**高频**使用的快捷键组合作为参考。注意本文不包含 macOS 系统通用的快捷键如 `Command + S 保存`。如果你想了解全部快捷键，可以前往查看 [Sketch 官方文档](https://www.sketch.com/docs/shortcuts/)。

## 视图
* Command + 1 预览所有图像
* Command + 2 放大选中图像至视图 100%
* Command + 3 将选中图像移动至视图中央
* Command + 0 显示实际尺寸
* Command + . 隐藏所有界面
* 按下 Space 移动光标可以移动视图

## 项目
* Command + N 新建项目
* Command + Shift + K 修改颜色配置文件

## 图像
* A 插入 Art board
* R O L V T 插入矩形、椭圆型、直线、矢量工具、文本
* Command + Shift + L 切换锁定图层
* Command + Shift + H 切换隐藏图层
* Control + P 切换像素模式
* Control + X 切换像素网格

## 编辑
* Command + 点击 选择组内对象
* Command + Shift + 点击 选择组内多个对象
* Command + Shift + R 旋转图像
* Control + C 取色计
* Command + Option + 拖动 拖动特定元素
* Command + K 等比缩放
* Command + Option + C 拷贝图层样式
* Command + Option + V 粘贴图层样式
* Command + [ 图层向前一层
* Command + ] 图层向后一层
* Command + G 图层编组
* Command + Shift + G 图层取消编组

## 视图
* Control + G 显示网格线
* Control + R 显示标尺
